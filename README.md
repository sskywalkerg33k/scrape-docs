# API Docs Scraper

Инструмент для автоматического сбора документации с портала разработчиков и конвертации её в единый Markdown-файл.

Этот файл удобно использовать как контекст для LLM (ChatGPT, Claude, Gemini) при написании кода для платформы.

## 📋 Требования

- Python 3.8 или выше
- `pip` (пакетный менеджер Python)

## 🚀 Установка и настройка

### 1. Подготовка проекта

Если вы скачали код или клонировали репозиторий, перейдите в папку проекта:
```bash
cd scrape-docs
```

### 2. Создание виртуального окружения

Рекомендуется использовать виртуальное окружение, чтобы изолировать зависимости проекта от системных.

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows:**
```powershell
python -m venv venv
.\venv\Scripts\activate
```

_После активации перед командной строкой должно появиться `(venv)`._

### 3. Установка зависимостей

Установите необходимые библиотеки (requests, beautifulsoup4, markdownify):
```bash
pip install -r requirements.txt
```

## 🛠 Использование

Запустите скрипт командой:
```bash
python scrape_docs.py
```

### Результат
После завершения работы скрипта в папке появится файл:
- `full_context.md` — содержит всю собранную документацию с разметкой.

Этот файл автоматически добавлен в `.gitignore`, чтобы не засорять репозиторий.

## ⚙️ Настройка списка URL

Чтобы добавить или удалить страницы для скачивания, откройте файл `scrape_docs.py` и отредактируйте список `urls` в начале файла:

```python
urls = [
    "https://dev.predict.fun/faqs-684248m0.md",
    # Добавьте новые ссылки сюда
]
```
